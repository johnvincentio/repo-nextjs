# Configure Ghost

```
home: www.johnvincent.io/
ghost blog: www.ghost.johnvincent.io/
```

Download or create a theme.

Copy it to

```
/var/www/ghost/content/themes
```

Restart Ghost

Ghost Dashboard

```
https://ghost.johnvincent.io/ghost
```

* Settings
* General
	* Blog Logo:
		* upload image
	* Blog Cover:
		* upload image
* Set time zone
* Set Facebook URL
* Set Twitter URL

* Activate your theme
* Save>

Navigation:

* Add navigation items as needed.
